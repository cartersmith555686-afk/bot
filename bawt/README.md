# Discord Moderation Bot + Free Dashboard

A fully-featured Discord moderation bot with:
- AutoMod
- Warnings
- Logging
- Plugin system
- Free Flask dashboard
- Discord OAuth2
- Replit-compatible

## Features
- Prefix & Slash commands
- Spam / Caps / Link protection
- Role-based AutoMod bypass
- Web dashboard controls
- SQLite persistence

## Setup
1. Clone repo
2. Install requirements
3. Add environment variables
4. Run `main.py`

## Environment Variables
TOKEN=
CLIENT_ID=
CLIENT_SECRET=
REDIRECT_URI=
